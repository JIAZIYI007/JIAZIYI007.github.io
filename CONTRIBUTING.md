# 提交建议/直接修改

您可以通过本仓库的[Issues页面](https://github.com/JIAZIYI007/JIAZIYI007.github.io/issues)提交建议,或将此仓库创建分支后直接修改并提交PR。
请确保您的PR与已有PR(包括开启状态和已关闭的PR)未重复后再提交。

#有关图片上传的问题
图片单个不得超过25MB，目前大多数与图床相关的PR请求都不会过审
我们会有文件上传页面，请勿通过PR上传。

# PR/Issues要求
## 不会处理的PR

- 进行大范围错误或误导性修改
- 破坏目录架构
- 删除.github目录
- 含违法内容
- 无效或重复
- 其它不合适的PR

## 不会处理的Issues
- 重复或无效
